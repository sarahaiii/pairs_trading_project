# Simple Pairs Trading Strategy Using Cointegration

## Project Description

This project builds a pairs trading strategy across S&P 500 stocks. The goal is to identify pairs of stocks whose prices are cointegrated, construct spread-based trading signals, and backtest a mean-reversion trading strategy.

## Why This Project Is Useful

This project teaches statistical arbitrage and combines finance, statistics, and trading strategy design. It also trains students to avoid overfitting by using out-of-sample testing.

## Where to Find Data

Historical stock price data can be collected from Yahoo Finance using the `yfinance` Python library.
